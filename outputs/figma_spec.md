# AR 互动软件 Figma 复刻规格

这份文档用于把当前 HTML 线框稿复刻到 Figma。目标不是做最终视觉稿，而是搭出结构清晰、可继续精修和做 prototype 的低保真 iOS 风格 UI。

## 画布与页面

建议在 Figma 中建立 3 个主 Frame：

| Frame 名称 | 尺寸 | 用途 |
| --- | --- | --- |
| `01 AR 互动面` | `390 x 844` | 默认 AR 相机页 |
| `02 焦距调节态` | `390 x 844` | 按住综合键并左右滑动时的临时状态 |
| `03 我的界面` | `390 x 844` | 用户个人页 |

外层展示用的白色卡片不是 App UI 的一部分，可以不做。真正需要复刻的是手机屏幕内部。

## 全局视觉

### 字体

优先使用 Figma 的 `SF Pro`。如果没有，用系统默认 San Francisco 或 Inter 替代。

| 用途 | 字号 | 字重 |
| --- | ---: | ---: |
| 大标题，例如“我的” | `32` | `700` |
| 卡片标题 | `17` | `600` |
| 底部标签 | `11` | `500` |
| 状态提示 | `13-14` | `600` |
| 辅助说明 | `12` | `400` |

不要把焦距数值做大，尤其是 `1x`，保持小而清晰即可。

### 颜色

| Token | 色值 | 用途 |
| --- | --- | --- |
| `iOS Blue` | `#007AFF` | 主要操作色，我的页按钮/任务图标 |
| `Text Primary` | `#1D1D1F` | 正文主文字 |
| `Text Secondary` | `#6E6E73` | 辅助文字 |
| `System BG` | `#F5F5F7` | 我的页背景 |
| `Dark Glass` | `rgba(28,28,30,0.34)` | AR 控件毛玻璃底 |
| `Light Line` | `rgba(255,255,255,0.46)` | AR 控件边线 |
| `Camera Dot` | `#30D158` | iOS 相机隐私绿点 |

### 毛玻璃效果

Figma 里可以这样设置：

- Fill: `#1C1C1E`，Opacity `34%`
- Stroke: `#FFFFFF`，Opacity `46%`
- Background Blur: `24`
- Drop Shadow: `0, 14, 30, 18% black`
- Inner Highlight: 复制一个 1px 白色上边线，Opacity `22%`

## 共用组件

### iPhone 屏幕壳

如果你只是做 App 内界面，可以不用画手机壳；如果要做提案展示，可以做：

- 外框：`390 x 844`
- Corner Radius: `48`
- Stroke: `9px #1C1C1E`
- Dynamic Island: `112 x 31`，黑色，圆角 `999`
- Home Indicator: `132 x 5`，圆角 `999`

### AR 相机背景

低保真阶段用渐变即可：

- Frame 填充：斜向渐变，近似 `#8EB5AE -> #C3D0CA -> #D0C5AA -> #8C957C`
- 增加一条浅色斜线/斜面，模拟地面或桌面平面
- 可以加一个很轻的中心 reticle：
  - 外圆：`110 x 110`
  - Stroke: 白色 `36%`
  - 中心十字线：白色 `42%`

## 01 AR 互动面

### 状态窗口

平常隐藏，只在 AR 状态异常或需要提醒时弹出。

隐藏态在稿子里可以保留一个低透明占位：

- 位置：顶部中间，约 `Y = 76`
- 尺寸：`190 x 42`
- 圆角：`999`
- Fill: Dark Glass
- Stroke: 白色 `46%`，可用虚线表达“默认隐藏”
- 文案：`状态窗口（平常隐藏）`
- Opacity: `28%`

弹出态文案示例：

- `移动手机以识别平面`
- `光线较暗，请靠近目标`
- `已识别平面，可放置物体`
- `追踪丢失，缓慢移动手机`

### 底部三段式控件

这是当前 AR 页最重要的交互组件，不是三个独立按钮。

容器：

- 位置：底部安全区上方，约 `X = 24, Y = 694`
- 尺寸：`342 x 96`
- 圆角：`30`
- Fill: Dark Glass
- Stroke: 白色 `46%`
- Background Blur: `24`
- 内部分成三等分
- 分割线：两条 `1px` 白线，Opacity `18%`

三段内容：

| 分段 | 图标建议 | 标签 | 说明 |
| --- | --- | --- | --- |
| 左 | 十字/罗盘/探索图标 | `探索` | 发现附近内容、推荐任务、扫描可互动对象 |
| 中 | 干净白色圆点 | `综合` | 添加物体、录制、拍摄的统一入口 |
| 右 | 用户圆点/头像入口 | `我的` | 进入个人中心 |

中间综合圆点：

- 外层不做复杂加号，不做红点
- 白色圆点直径约 `42`
- 外部可以加一圈暗色阴影，模拟拍摄按钮质感
- 标签仍写 `综合`
- 扩展功能不要塞在图标上，放到新手引导中解释

### 新手引导建议

第一次进入 AR 页时，围绕综合按钮弹出一层轻引导：

- 轻点：拍摄
- 长按：录制
- 上滑或展开：添加物体
- 按住上方区域并左右滑动：调节焦距

这部分可以后续作为 `04 新手引导态` 单独画。

## 02 焦距调节态

触发方式：用户按住综合按键上方屏幕区域并左右滑动。

### 页面变化

- AR 背景保持可见
- 底部三段式控件保留，但透明度降到约 `72%`
- 出现大半径圆形焦距盘
- 松手后焦距盘消失

### 顶部提示

- 文案：`按住并左右滑动调节焦距`
- 位置：顶部中间，约 `Y = 86`
- 尺寸：`222 x 43`
- 圆角：`999`
- Fill: Dark Glass
- Stroke: 白色 `48%`
- Background Blur: `22`

### 大圆焦距盘

重点：它应该看起来是一个“很大的圆盘只露出上部一段”，不是横向滑条。

建议做法：

1. 画一个大圆，直径约 `470-520`
2. 圆心放在屏幕下方，低于底部三段式控件
3. 只让上方约 `2/5` 露出
4. 外圆 Stroke 用白色 `45-52%`
5. 内圆辅助线用白色 `28%`
6. 两侧用透明遮罩或降低透明度，让注意力集中在中间

当前视觉关系：

- 外圆半径大，圆心偏下
- `1x` 在中间小气泡里
- `0.8x`、`1.5x` 在左右两侧，透明度降低并轻微模糊
- 更远的 `0.5x`、`2x` 更淡、更虚

### 焦距数字

| 倍数 | 视觉 |
| --- | --- |
| `1x` | 中心选中，小气泡，字号约 `14` |
| `0.8x / 1.5x` | 左右次级，字号 `12`，Opacity `42%` |
| `0.5x / 2x` | 更远层级，Opacity `20%`，轻微 blur |

注意：不要把 `1x` 气泡做大。它只是选中标记，不是主按钮。

### 刻度

刻度应放在圆的外半径上：

- 使用 9 根左右的短竖线
- 中心刻度最长、最清楚
- 离中心越远，Opacity 越低，Blur 越强
- 中心刻度可设 `2px` 宽，其他 `1px`

## 03 我的界面

### 背景与标题

- 背景：`#F5F5F7`
- 顶部 padding：约 `62`
- 标题：`我的`，`32px / 700`
- 右上设置按钮：`40 x 40`，圆形，浅白背景

### 个人信息卡

- 尺寸：宽度约 `354`，高度自适应
- 圆角：`18`
- Fill: `rgba(255,255,255,0.86)`
- Stroke: `rgba(60,60,67,0.08)`
- 左侧头像：`64 x 64`，圆角 `22`
- 文字：
  - `用户名 / 昵称`
  - `等级、成就、最近一次 AR 互动时间`

### 自定义角色卡

- 高度：`188`
- 圆角：`20`
- 背景：浅蓝到浅橙的弱渐变
- 右侧卡牌：
  - 前卡 `124 x 144`
  - 圆角 `18`
  - 轻微旋转 `-5°`
  - 背后加一张虚线卡牌，旋转 `7°`
- 交互：可在 Prototype 中做 `Smart Animate` 卡牌翻转

### 放置过的任务

每个任务项：

- 高度：约 `62`
- 圆角：`16`
- Fill: `rgba(255,255,255,0.86)`
- 左侧图标：`42 x 42`
- 主标题：任务名
- 副标题：状态，例如 `已放置 · 可继续互动`
- 右侧：`进入` / `回看`

### 底部导航

我的页可保留普通底部导航：

- 高度：`58`
- 圆角：`28`
- 三项：`AR / 记录 / 我的`
- 如果产品后续决定 AR 页也用三段式底部控件，可以把我的页底部也统一成同一组件。

## Prototype 交互建议

### AR 互动面

| 触发 | 结果 |
| --- | --- |
| 点击 `探索` | 打开探索内容面板或切到探索状态 |
| 点击 `我的` | 跳到 `03 我的界面` |
| 点击 `综合` | 执行拍摄或打开快捷动作 |
| 长按 `综合` | 录制 |
| 上滑 `综合` | 添加物体 |
| 按住综合键上方并左右滑动 | 显示 `02 焦距调节态` |

### 焦距调节态

| 触发 | 结果 |
| --- | --- |
| Drag left/right | 焦距数值切换，圆盘刻度跟随移动 |
| Release | 回到 `01 AR 互动面` |

### 我的界面

| 触发 | 结果 |
| --- | --- |
| 点击角色卡 | 卡牌翻转 |
| 点击任务 | 回到对应 AR 场景 |
| 点击设置 | 打开设置页 |

## 下一步精修方向

- 把 `探索 / 综合 / 我的` 做成一个 Figma component，之后用 variants 表达选中态。
